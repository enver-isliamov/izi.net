# UX Evolution Plan — izinet

Этот план направлен на упрощение пользовательского опыта (UX) и улучшение визуальной иерархии (UI) проекта без изменения текущего функционала.

## Статус: ✅ Завершено 

---

## Этап 1: Упрощение Навигации и Базовая Консолидация (Завершен ✅)
**Цель:** Снизить когнитивную нагрузку и количество кликов для доступа к основным функциям.

- [x] **1.1. Реорганизация Админ-панели:**
    - [x] Объединены страницы "Статистика" и "Серверы".
    - [x] Навигация в админке обновлена (Обзор/Серверы, Пользователи, Платежи, Настройки).
    - [x] Финальная полировка дашборда админа: компактные карточки серверов и быстрая статистика.
- [x] **1.2. Реорганизация Личного Кабинета Пользователя:**
    - [x] Объединение "Главной" и "Подписки" в одну страницу "Обзор" (Home).
    - [x] Упрощенный сайдбар: Дом, Инструкции, Профиль, Рефералы.
- [x] **1.3. Унификация Стиля Таблиц:**
    - [x] Единый стиль мобильных карточек для всех табличных данных (пользователи, транзакции, серверы).

## Этап 2: Визуальная Иерархия и Управление Устройствами (Завершен ✅)
**Цель:** Сделать управление VPN-ключами интуитивно понятным.

- [x] **2.1. Карточки Устройств (Пользователь):**
    - [x] Акцентные кнопки действий (QR-код, Скопировать, Продлить) вместо текстовых ссылок.
    - [x] Улучшенные индикаторы статуса (Online/Offline/Expired) с неоновым свечением.
- [x] **2.2. Улучшение Визарда Подписки:**
    - [x] Объединены выбор периода и класса сервера на один экран (Step 1: План).
    - [x] Визуальный акцент на итоговой стоимости в реальном времени.
    - [x] Сокращение количества шагов с 5 до 4.

## Этап 3: Компактность и Микро-взаимодействия (Завершен ✅)
**Цель:** Сделать интерфейс более "живым" и быстрым.

- [x] **3.1. Оптимизация Модальных Окон:**
    - [x] Переход от тяжелых модальных окон к боковым панелям (Sheet/Slide-over) для истории операций.
- [x] **3.2. Анимации и Состояния:**
    - [x] Удалены тяжелые анимации переходов страниц для мгновенного отклика (по запросу).
    - [x] Внедрены Skeleton Loaders для Dashboard, Wallet и Admin/Users для мягкой визуальной загрузки данных.
    - [x] Сохранены легкие микро-анимации наведения (Hover) на карточках для интерактивности.
