ТЕХНИЧЕСКОЕ ЗАДАНИЕ
VPN Subscription Service — "izinet"
1. ОБЩЕЕ ОПИСАНИЕ
1.1 Назначение системы
Веб-сервис для управления VPN-подписками с личным кабинетом пользователя, системой оплаты и автоматической выдачей конфигураций.

1.2 Референс
Дизайн и функционал заимствованы с izinet

1.3 Инфраструктура заказчика
VPN Server IP: 185.72.11.57
├── 3x-ui Panel (порты 443, 8443)
│   ├── VLESS + Reality (443) — основной
│   └── VLESS + WS + TLS (8443) — VIP
├── Nginx (80) — заглушка Pomodoro
├── MTProto Proxy (9443) — Docker
└── AdGuard DNS
1.4 Планируемый стек
Frontend: React + TypeScript + TailwindCSS → Vercel
Backend: Supabase (Auth + Database + Edge Functions)
Репозиторий: GitHub
Платёжная система: Enot.io
